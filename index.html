Login flow setup (Power Automate)
  • Create a new Automated cloud flow → trigger: "When a HTTP request is received"
  • JSON schema:
    {"type":"object","properties":{"loginId":{"type":"string"}}}
  • Add action: Excel Online → Get a row (or "List rows present in a table") → ModInfo table → filter: loginId eq @{triggerBody()?['loginId']}
  • Add action: Response → Status 200 → Body:
    {"found": true, "modId": "@{first(outputs('List_rows')?['body/value'])?['modId']}", "modName": "@{first(outputs('List_rows')?['body/value'])?['modName']}", "location": "@{first(outputs('List_rows')?['body/value'])?['location']}", "email": "@{first(outputs('List_rows')?['body/value'])?['email']}", "phone": "@{first(outputs('List_rows')?['body/value'])?['phone']}"}
  • If no row found, add a parallel Response with body:
    {"found": false}
  • Save → copy HTTP POST URL → paste above
TaskData sync setup guide
  • Go to make.powerautomate.com → Create → Automated cloud flow → Skip the dialog.
  • Click Add a trigger → search "HTTP" → select "When a HTTP request is received" (Request connector).
  • Paste the JSON schema into the Request Body JSON Schema field:
    {"type":"object","properties":{"taskId":{"type":"integer"},"taskTitle":{"type":"string"},"category":{"type":"string"},"distance":{"type":"string"},"duration":{"type":"string"},"luxReading":{"type":"string"},"status":{"type":"string"},"stepsDone":{"type":"integer"},"stepsTotal":{"type":"integer"},"participantId":{"type":"string"},"moderator":{"type":"string"},"sessionDate":{"type":"string"},"deviceId":{"type":"string"},"completedAt":{"type":"string"},"notes":{"type":"string"},"submittedAt":{"type":"string"},"modId":{"type":"string"}}}
  • Add a step: Excel Online (Business) → "Add a row into a table" → point it at your Excel file and the table named TaskData. Map each column to its dynamic value.
  • Save the flow → copy the HTTP POST URL from the trigger card → paste it above.
